# Kindle Agent

Claude Code agent that formats and sends documents, transcripts, and text content to Kindle via Gmail SMTP. Supports PDF, MOBI, EPUB, DOCX, TXT, HTML, and raw text -- automatically cleans and converts content for optimal Kindle reading with reflowable HTML.

## Usage

This is a Claude Code agent. Add the `.md` file to your `.claude/agents/` directory to use it.
